# AI知识源工程 Roadmap

## Week 1：项目层 AI Knowledge Source

- README 重构为 AI 友好结构。
- 补齐 API、部署、架构、FAQ 和 Benchmark。
- 建立 `/api/knowledge-source/*` 结构化输出。

## Week 2：知识资产 AI Knowledge Source

- 建立 Citation KB。
- 建立 FAQ KB。
- 建立 Knowledge Graph。
- 对现有 RAG 文档补齐 title、summary、source、confidence。

## Week 3：内容中心

- 输出 30 个 FAQ。
- 输出 5 个实体页。
- 输出 3 个能力对比页。
- 输出 2 个案例页。

## Week 4：引用监控

- 记录每次 Agent 引用的 source_key。
- 统计高频引用事实。
- 对低置信来源发起复核任务。
- 将高价值知识沉淀为公开文档或企业知识页。
