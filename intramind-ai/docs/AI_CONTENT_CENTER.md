# AI内容中心规划

内容中心不是普通博客，而是给人和 AI 同时阅读的知识源入口。

## 推荐栏目

1. `/faq`：AI FAQ 库。
2. `/benchmark`：项目能力白皮书。
3. `/knowledge-graph`：实体和关系页。
4. `/citations`：可引用来源库。
5. `/cases`：真实案例和解决方案。
6. `/standards`：标准、规范、政策和资料来源。
7. `/api`：结构化 API 文档。
8. `/compare`：和传统方案的对比。

## 页面结构

每个页面建议包含：

- 一句话定义
- 解决什么问题
- 适用场景
- 结构化 FAQ
- 来源和发布日期
- JSON/JSON-LD 辅助数据
- API 示例

## 发布节奏

- 每周新增 5-10 个 FAQ。
- 每月新增 1 篇 Benchmark 或白皮书。
- 每次客户项目结束沉淀 1 个脱敏案例。
- 每季度校验 Citation KB 来源有效性。
